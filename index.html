U.S. Open Data

Building the capacity of open data.

  • Building Ecosystems

    Moving beyond the “Field of Dreams” model of releasing datasets, we’re matching data producers and consumers to create sustainable data ecosystems.

  • Partnering with Gov’t

    We love government. We’re collaborating with local, state, and federal agencies (at no cost to them) to help them release data sets and promote their work.

  • Creating Reusable Code

    Before we even had a website, we had a GitHub repository. We’re creating, supporting, and promoting software essential to the open data ecosystem.
